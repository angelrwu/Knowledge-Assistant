Welcome this is the knowledge assistant.

The reason for this website is to make an enviroment where we can decrease the delta in the current learning absorption of information. We will be doing this by making an open enviroment to test, creat and make easy learning aids from existing class notes. Thus this website will focus on the following:

  • Additive or specialized content creation
  • Content verification through crowdsourcing
  • Easy learning aid creation and analysis
